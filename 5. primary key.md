## create table & primary key

```
create table table_name(
columen_name_1 type constraints,
...
);
```
- for example:

```
order_id int not null primary key auto_increment,
```

- or we can assign the primary key at the end
```
create table table_name(
order_id int,
...

primary key(order_id)
);
```
